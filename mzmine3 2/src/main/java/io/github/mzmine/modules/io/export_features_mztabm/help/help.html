mzTab-m export

Description

This module exports the contents of the peak list into a mzTab-m file, which can later be processed by other tools.

Method parameters

Peak lists (input)
Name of peak list to be exported selected in Main Window
Filename
Name of file where the exported data is saved
Include all peaks
If selected, peaks with unknown identity will also be exported to the mzTab-m file
Miscellaneous Info
  • Fow importing study variable, user parameters are checked, with whichever first parameter name contains study variable is used and rest are ignored.
