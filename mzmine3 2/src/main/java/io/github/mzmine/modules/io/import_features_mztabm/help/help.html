mzTab-m import

Description

This module imports the contents of a mzTab-m file into a peak list. If links to raw data files are available in the mzTab-m file, these can automatically be imported together with the peak list. The raw data import looks for files in the location provided in the mzTab-m file, if not present it looks in the same folder as that of mzTab-m file, if still not present it looks if files are present in compressed form (.gz or .zip). If files are nowhere to be found finally empty rawFiles are created.

Method parameters

File
Name of mzTab-m file to be imported. Add "{}" to the file name to export multiple features, "{}" will be replaced by be replaced by the name of feature list.
Import raw data files?
If selected, raw data files will also be imported if they are available. Files will be loaded from the msrun location defined in the mzTab-m file or from the same folder containing the mzTab-m file. If the raw data files are not available, empty raw data files will be generated by MZmine. Please note that missing files may lead to issues with other modules that need access to scan data.
